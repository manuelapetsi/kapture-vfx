GUIDE

Studio Workflow

A short operating sequence for repeatable results.

  1. Start the session and allow the browser to access your camera.
  2. Clear the scene, then reset the background once the frame is clean.
  3. Sample the cloak color directly from the live feed or set it manually.
  4. Adjust the mask controls until the preview isolates only the cloak region.
  5. Disable preview to return to the final invisibility render.

Keep the subject separated from similarly colored objects for the cleanest result.
